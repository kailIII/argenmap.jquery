Argenmap

Un plugin de jQuery para poner un mapa de Argentina en tu web.

argenmap-jquery te permite usar un mapa de Argentina en tu sitio, con los datos del Instituto Geográfico Nacional de Argentina.

  • Indicá una ubicación con un marcador simple
  • Superponé tus propios datos
  • Visualizá servicios WMS sobre un mapa del IGN

argenmap es un servicio web del Instituto Geográfico Nacional de la República Argentina.

El código de argenmap.jquery está disponible bajo una licencia basada en la BSD.

La documentación de argenmap.jquery usa Bootstrap, de Twitter y la librería jQuery y la generamos con phrozn. El hosting del proyecto corre por cuenta de Github.
