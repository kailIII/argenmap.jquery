Tutorial! Migración de un mapa de Google Maps a uno de argenmap

Migrar desde Google Maps a Argenmap.jquery

Este es un ejemplo basico de uso de Google Maps: 

    <script type="text/javascript" >
    var map;
    function initialize() {
    var mapOptions = {
    zoom: 8,
    center: new google.maps.LatLng(-37.9800587, -57.58995909999999),
    mapTypeId: google.maps.MapTypeId.ROADMAP
    };
    map = new google.maps.Map(document.getElementById('mapahtml'),
    mapOptions);
    }

    google.maps.event.addDomListener(window, 'load', initialize);
    </script>

Los valores que tenemos que tomar para poder migrarlos a argenmap.jquery son los siguientes:

    <script type="text/javascript" >
    var map;
    function initialize() {
    var mapOptions = {
    zoom: 8,
    center: new google.maps.LatLng(-34.5722189, -58.44053759999997),
    mapTypeId: google.maps.MapTypeId.ROADMAP
    };
    map = new google.maps.Map(document.getElementById('mapahtml'),
    mapOptions);
    }

    google.maps.event.addDomListener(window, 'load', initialize);
    </script>

Los atributos se encuentran subrayados con linea de puntos, para una mayor comprensión se listan aqui:

  • zoom = 8
  • centro = -37.9800587, -57.58995909999999
  • elementId = mapahtml

Convirtiendo el ejemplo de Google Maps a Argenmap.jquery usando el servicio WMS del Instituto Geográfico Nacional

Siguiendo con el ejemplo anterior ahora veremos como instanciar el mapa de argenmap.jquery con los parametros que tomamos

    $(function () {
    $('#mapahtml').argenmap().centro(-37.9800587, -57.58995909999999).zoom(8);
    });
Acerca el servicio WMS del Instituto Geográfico Nacional a los diseñadores de páginas de instituciones públicas y permite ampliar la presencia de los mapas nacionales en la web.
