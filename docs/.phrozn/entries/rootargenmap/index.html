Datos oficiales

Argenmap permite mostrar un mapa de Argentina en un sitio web con los datos del Instituto Geográfico Nacional de Argentina.

Al estilo jQuery

Argenmap es un servicio web orientado a diseñadores y programadores web.

argenmap-jquery es una librería de mapas web diseñada en forma de plugin de jQuery así que podés esperar todo lo que esperarías de los plugins que ya conocés.

Calidad del servicio

Importante: Se informa al usuario que al implementar este servicio en su propio sitio web, se requiere informar al Instituto Geográfico Nacional acerca de esta demanda para su evaluación a fin de poder ofrecer la calidad de servicio requerida. Favor de enviar solicitud a argenmap@ign.gob.ar.
