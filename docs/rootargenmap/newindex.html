argenmap Un plugin de jQuery para poner un mapa de Argentina en tu web.

Así se puede ver un mapa de Argentina, con los datos del Instituto Geográfico Nacional en tu sitio.

Mirá este ejemplo en una página aparte » (mapasimple.html).

Audiencia

Esta sección está diseñada para gente familiarizada con el lenguaje de programación Javascript y con la librería jQuery.

Cómo usar un mapa en tu sitio ».

Requerimientos

  • argenmap.jquery necesita tanto la librería jQuery como la librería de Google Maps API.
  • La versión mínima de jQuery que necesita es 1.7.2 pero se recomienda la última versión.

Cómo empezar

La guía para usuarios describe argenmap con ejemplos, detalles y la explicación de sus características tanto para programadores como para diseñadores web y usuarios del mapa.

Datos del mapa

Argenmap es un sistema de información geográfica con datos de origen mixto.

  • Podés leer más acerca de los datos que muestra un mapa hecho con argenmap.jquery en Datos del mapa.

  • IGN Argentina

    Datos oficiales de la República Argentina

  • OpenStreetMap

    Datos Libres del proyecto OpenStreetMap

  • Google Satellite

    imágenes de satélite provistas por Google

argenmap es un servicio web del Instituto Geográfico Nacional de la República Argentina.

El código de argenmap.jquery está disponible bajo una licencia basada en la BSD.

La documentación de argenmap.jquery usa Bootstrap, de Twitter y la librería jQuery y la generamos con phrozn. El hosting del proyecto corre por cuenta de Github.
